TextSpam

HeartRateMonitor

Measure your heart rate anytime, anywhere.



Using some clever image processing algorithm, HeartRateMonitor accurately tracks one's heart rate and creates a moving average with sampled data. This data is then graphed with respect to time to show trends in the rate over a short period of time. (Created for Biology SBI4U)

TextMetrics

Analyze your texting habits.



TextMetrics creates a visual representation of who you text most often. The application scans all of a users messages and then displays a pie chart with a breakdown with respect to how many messages are sent to and from each contact.(Created for Computer Science ICS4U)

MyoMove

A gesture suite for the Myo wearable device.



MyoMove adds several useful features to the Myo through using several easy to perform gestures. MyoMove can control music playback as well as the lock state of the phone. The application uses a combination of gesture tracking on the Myo and accelerometer data from the phone to determine whether or not to unlock the phone.(Created for Hack the North 2014)

Hive

A Social Gathering Application



Hive lets you see nearby events from a crowdsourced database. When the backend is running, the service allows user to submit events to the central database and for any user to see these events if they are withing a certain radius of the event geographically. (Created for YHack 2014)

SecretSauce

An anonymous Dropbox.



SecretSauce is a dropbox alternative, presenting shared files in way similar to reddit. The application centralized file sharing with a emphasis on privacy; there are no user accounts, no cookies, and no traces of a user having used the service. All for the sake of privacy. The application utilizes flask and can be run on practically any network connected system. (Created for SEHackDay 2014)

ScribblerPlaysTwitch

A Multiplayer Collaborative gaming experience.



ScribblerPlaysTwitch is exactly what the name suggests. Twitch plays but for the a scribbler robot, all running on a fluke board. Using various web technologies, such as sockets, users can communicate with one another and submit commands to the robot from anywhere in the world. They can also get a live video feed from the robot as well as from one more webcam to allow the users to get a good overview of the surrounding of the robot. The project also utilizes OpenCV for some of the image processing tasks. Users can play with anyone around the world and control the scribbler robot to complete various tasks. With a seamless interface coupled with exciting gameplay you'll be "Hooked" on the "Feeling" that the game gives you.
